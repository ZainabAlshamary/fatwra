Get Starterd


The Rest API is the main way to get data into and out of fatwra platform. It's an HTTP-based API that apps can use to programmatically query data, create new bills, manage bills, create services, and perform a wide variety of other tasks.

First to use these APIs, you need to obtain a JWT Token, then you can use this Token as a Bearer Authorization in each request.


Environment details

Environment: UAT
Base API URL: https://api-uat.fatwra.net


Environment: Production
Base API URL: https://api.fatwra.net

Main APIs

Login

Using Token API to obtain a JWT Token and using it as header Bearer Authorization in other APIs

Parameters Description Status Type
Username Accept Phone Number, Email, Username Required String
Password Accept Numbers and Characters Required String
HTTP Request
POST {{BASE_URL}}/v1/api/Auth/Token
Headers
Content-Type: application/json
PayLoad
{
  "username": "your-login-info@fatwra.net",
  "password": "xxxxxxxx"
}

HTTP Request
PUT {{BASE_URL}}/v1/api/Provider/AddBill
Headers
Content-Type: application/json
Authorization: Bearer {{YOUR_TOKEN}}

Search Bill

This API for getting bills where matching one of the following:
PayId, Customer Phone Number or the Note Custome Field.
Parameters Description Status Type
searchTerm Bill PayId, Customer PhoneNumber, Note Required Querey Parameter
start Pagination (Get bills from index) Required Querey Parameter
start Pagination (Take bills count) Required Querey Parameter
HTTP Request
GET {{BASE_URL}}/v1/api/Provider/SearchBill?searchTerm=xxxxxxxxxxxxxxxx&start=0&end=10
Headers
Content-Type: application/json
Authorization: Bearer {{YOUR_TOKEN}}

None.
